Here is an efficient way to calculate Fibonacci numbers using memoization:
fibonacci.py 
def fib(n, memo={}):
    if n in memo: return memo[n]
    if n <= 2: return 1
    memo[n] = fib(n-1, memo) + fib(n-2, memo)
    return memo[n]
